Omniauth for [SUZURI](https://suzuri.jp/)

Instration
----------

Add this line to your application's Gemfile:

    gem 'omniauth-suzuri'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omniauth-suzuri

Test
----

```
bundle exec rspec
```
